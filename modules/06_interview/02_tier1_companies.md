## 2.1 字节跳动

### 公开面经线索

| 线索 | 平台 | 备注 |
|------|------|------|
| `字节大模型算法岗三面面经` | 新浪公开文章 | 标题里直接出现 `Qwen / 手撕 / 技术面` |
| `字节大模型Agent算法面经` | 新浪公开文章 | 直接带 `Agent` 关键词 |
| `字节大模型算法面经` | 牛客 | 偏算法与实习方向 |
| `字节大模型面经合集(二)` | 牛客 | 更像多轮追问汇总 |
| `字节大模型面经(一面)` | 牛客 | 一面基础与项目结合 |

### 高频知识点

`RAG 排查` `Agent 工具调用` `Qwen / 开源模型选型` `线上链路分析` `手撕 + 场景题`

### 代表题目

| 题目 | 面试官更想听什么 | 对应通用题库 |
|------|------------------|--------------|
| 你们的 RAG 效果不好，按链路怎么排查？ | 先排检索还是先排生成、指标、回归集 | `#94` `#97` `#98` `#105` `#114` |
| Tool Description 为什么会明显影响结果？ | 工具设计而不是只怪模型弱 | `#122` `#123` `#135` |
| 给一个现有业务，什么时候该上 Agent，什么时候不该上？ | 业务边界、ROI、失败恢复 | `#119` `#121` `#133` |

### 风格提醒

- 第一问不一定难，**真正的压力在连续追问**
- 很喜欢从“你做过什么”切到“如果线上效果差你怎么排”
- 回答时最好带一个真实业务例子，不要只讲定义
