## 2024 及更早：基础盘决定你能不能接住追问

| 热点 | 为什么仍然重要 | 优先题目 |
|------|----------------|----------|
| Transformer 基础 | 任何新架构几乎都离不开它 | `#1` `#3` `#14` `#31` |
| LoRA / QLoRA | 微调岗位长期高频 | `#69` `#70` `#72` |
| 经典 RAG Pipeline | 很多公司至今还在用 | `#94` `#98` `#103` |
| ReAct / Function Calling | Agent 基础形态 | `#120` `#122` `#123` |
| 评估与安全基本盘 | 热点会变，但安全与评测不会消失 | `#162` `#165` `#168` |

### 这部分最容易被忽略的问题

- **为什么 Attention 要除以 `√d_k`？**
- **为什么向量检索对专有名词和精确数字不友好？**
- **LoRA 的 rank 为什么不是越大越好？**
- **为什么 `LLM-as-Judge` 不能直接当唯一标准？**
