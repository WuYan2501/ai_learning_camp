## 1.1 数学基础：目标就是先有感觉

> 这里的数学不是为了考试，而是为了看懂模型里的几个核心动作：怎么表示信息、怎么比较相似度、怎么衡量误差、怎么把参数一点点调好。
>
> **第一次读到这里，只要先回答一个问题就够了：这些数学概念，分别在模型里“干什么、管什么”。**

- **线性代数**
  - *先理解什么*：向量和矩阵怎么表示信息，模型怎样做“比较、投影、压缩”
  - *先记住这些词*：矩阵乘法、转置、SVD
  - *为什么重要*：Attention 本质上就是大规模矩阵运算；SVD 背后对应的是“高维信息能不能压缩”这个问题

- **概率统计**
  - *先理解什么*：模型为什么经常输出“概率”，以及“不确定”怎么表达
  - *先记住这些词*：条件概率、最大似然估计（MLE）、KL 散度
  - *为什么重要*：模型训练很多时候其实都在回答“怎样让预测结果更接近真实分布”

- **微积分**
  - *先理解什么*：参数改一点，loss 为什么会变化；模型为什么能顺着梯度往前走
  - *先记住这些词*：梯度、偏导数、链式法则
  - *为什么重要*：反向传播本质上就是把误差一层层往回传，再决定每个参数该怎么改

- **信息论**
  - *先理解什么*：模型怎样衡量“自己离标准答案还有多远”
  - *先记住这些词*：熵、交叉熵、KL 散度
  - *为什么重要*：交叉熵是语言模型里最常见的损失函数之一，后面会反复碰到

- **最优化**
  - *先理解什么*：参数这么多，模型为什么还能一点点往更好的方向走
  - *先记住这些词*：目标函数、约束、最优解
  - *为什么重要*：训练模型本质上就是一个“不断试着把结果调得更好”的过程

> 💡 **常见面试问法**
>
> - SVD 分解在 LoRA 中扮演什么角色？为什么 LoRA 不直接做 SVD？⭐
> - KL 散度不对称意味着什么？KL(P||Q) 和 KL(Q||P) 的区别在 RLHF 中有什么影响？⭐
> - 为什么 Cross-Entropy 能衡量模型输出和真实分布的差距？它和 KL 散度的关系？⭐
> - 梯度消失和梯度爆炸的根本原因是什么？在 Transformer 中是怎么解决的？
> - 为什么 Softmax + Cross-Entropy 是标配？能不能用其他组合？

## 1.2 机器学习核心：知道模型在解决什么问题

> 如果说数学是在讲“底层规律”，那机器学习这部分更像是在回答：**模型怎么学、学成什么样、怎么判断它学得好不好。**

- **监督学习算法**
  - *需要知道*：给定输入和标准答案，模型如何从样本里学出规律
  - *常见代表*：线性回归、逻辑回归、决策树、随机森林、XGBoost
  - *先有个感觉*：有些模型更像“画一条合适的线”，有些更像“不断做判断分支”，有些更像“很多弱模型一起投票或接力修错”

- **无监督学习**
  - *需要知道*：没有标准答案时，模型怎么自己发现数据结构
  - *常见代表*：聚类、降维、可视化
  - *为什么重要*：比如你拿到一堆 Embedding，往往要先看它们是不是自然分群、能不能被压到二维后看出结构

- **损失函数** ⭐
  - *需要知道*：模型怎么知道自己“错了多少”
  - *常见代表*：MSE（回归）、Cross-Entropy（分类/语言模型）
  - *核心问题*：为什么语言模型用 Cross-Entropy 而不是 MSE？因为语言模型预测的是“下一个 token 的概率分布”，不是一个单独数值

- **优化器** ⭐
  - *需要知道*：模型知道自己错了之后，参数应该怎么改
  - *常见演进*：SGD → Momentum → Adam → AdamW
  - *核心问题*：Adam 为什么通常收敛更快？AdamW 又为什么比 Adam 更适合现代大模型训练？

- **正则化**
  - *需要知道*：怎么避免模型只会背训练集，不会真正泛化
  - *常见方法*：L1 / L2 / Dropout / Early Stopping
  - *核心问题*：L1 和 L2 分别在限制什么？Dropout 为什么能减少过拟合？

- **偏差-方差权衡（Bias-Variance Tradeoff）** ⭐
  - *需要知道*：模型太简单会学不会，太复杂又容易学偏
  - *核心思想*：模型复杂度增加 → 偏差降低但方差升高；过拟合 = 高方差，欠拟合 = 高偏差
  - *延伸*：大模型时代，这个规律并没有失效，但表现形式比传统机器学习更复杂

- **评估指标**
  - *需要知道*：不能只看一个指标就觉得模型“不错”，很多时候要结合任务背景一起判断
  - *常见指标*：Accuracy / Precision / Recall / F1
  - *经典例子*：医院癌症检测，100 个人里 99 个人正常、1 个人患病。如果模型把 100 个人都判成正常，那 Accuracy 可以有 99%，但这个模型显然没法用

> 💡 **常见面试问法**
> - 为什么 Adam 容易过拟合？AdamW 是怎么解决的？⭐
> - 双降（Double Descent）现象是什么？⭐
> - 什么是过拟合？参数量远大于数据量的 LLM 为什么没有严重过拟合？⭐

## 1.3 深度学习基础：开始接近现代大模型

> 深度学习可以理解成从“传统机器学习”走向“神经网络世界”的桥梁。后面你看到的 Transformer、RAG、Agent，很多底层训练直觉都从这里延伸出来。
>
> **如果你只想先抓大模型最相关的内容，优先看：激活函数、归一化、训练技巧。**

- **激活函数** ⭐
  - *需要知道*：网络为什么不能只是线性堆叠，以及非线性到底带来了什么
  - *常见演进*：ReLU → GELU → SiLU
  - *直白一点说*：如果没有激活函数，很多层叠在一起，本质上还是在做线性变换，模型表达能力会很弱
  - *核心问题*：GELU 为什么比 ReLU 在 Transformer 中表现更好？
  - *延伸*：SwiGLU 是现代大模型 FFN 里很常见的结构（LLaMA、DeepSeek 都在用）

- **反向传播**
  - *需要知道*：模型到底是怎样把“错误”一层层传回去改参数的
  - *核心思想*：前向先算结果，反向再把误差往回传，决定每一层参数应该怎么改
  - *面试高频*：手推一个简单网络的反向传播

- **CNN 核心概念**
  - *需要知道*：为什么视觉模型早期几乎都绕不开卷积
  - *为什么重要*：CNN 虽然不是今天大模型的主角，但“局部感受野”“参数共享”“残差连接”这些思想仍然非常重要

- **RNN / LSTM / GRU**
  - *需要知道*：Transformer 出来之前，序列模型是怎么处理“前后文”的
  - *核心思想*：序列建模，隐状态负责把前面的信息往后带
  - *核心问题*：为什么 RNN 容易出现梯度消失？LSTM 的门控机制是怎么缓解的？
  - *历史地位*：理解它们，才能更容易理解 Transformer 为什么会带来代际变化

- **归一化方法** ⭐ 🔥
  - *需要知道*：归一化的核心作用就是让训练过程更稳，不容易数值失控
  - **BatchNorm**：更依赖 batch 统计量，训练和推理时行为不同
  - **LayerNorm**：对单个样本做归一化，不依赖 batch size，特别适合 NLP 和变长序列
  - **RMSNorm**：可以理解成 LayerNorm 的轻量版本，现代大模型里很常见
  - *核心问题*：为什么 LLM 更常用 LayerNorm / RMSNorm，而不是 BatchNorm？
  - *核心问题*：Pre-Norm vs Post-Norm，哪个训练更稳定？（Pre-Norm 更稳，现代 LLM 更常见）⭐

- **训练技巧**
  - *需要知道*：训练跑不稳、loss 发散、显存不够时，通常先从这几项排查
  - **学习率调度**：Warmup → Cosine Annealing
    - *核心问题*：为什么 LLM 训练通常都要先 Warmup？因为一开始参数还是随机的，直接上大学习率很容易不稳定
  - **梯度裁剪（Gradient Clipping）**：防止梯度突然过大，把训练直接带崩
  - **混合精度训练（AMP）** ⭐：常见是 BF16/FP16 做前向，FP32 保证参数更新更稳定
    - *核心问题*：为什么现在大模型训练更偏向 BF16？因为它更不容易溢出
  - **梯度累积**：显存不够时，用多次小 batch 累起来，模拟更大的 batch
  - **权重初始化**：好的初始化能让训练更顺利地起步，不容易一开始就数值不稳定

> 💡 **常见面试问法**
> - Transformer 为何选择 LayerNorm？—— 从 BatchNorm 的缺陷到 LN 的必然性 📝 ⭐

> - Transformer中什么是归一化，为什么要归一化？
> - Transformer 中 Pre-Norm 和 Post-Norm 是什么？
> - Pre-Norm 训练更稳定，但 Post-Norm 理论上上限更高，怎么理解这个矛盾？⭐
> - GELU 的数学形式是什么？为什么它比 ReLU 在 Transformer 中表现更好？⭐
> - SwiGLU 的 "Gate" 在做什么？为什么门控线性单元比普通 FFN 效果好？⭐
> - Dying ReLU 问题是什么？LeakyReLU/PReLU 是怎么解决的？为什么 LLM 不用？

> - Cosine Annealing 为什么比 StepLR 效果好？直觉上怎么理解？
> - BF16 比 FP16 动态范围大但精度低，为什么 LLM 训练反而更好？⭐
