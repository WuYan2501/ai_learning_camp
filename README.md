# 🗺️ AI工程地图

一份持续更新的 AI 学习与落地知识地图。

这里不会只堆零散资料，而是尽量把 **基础认知、模型原理、RAG、Agent、MCP、工程落地、实操技能、面试整理** 串成一条能走通的路径。

如果你是从知乎来的，这里可以直接当作资料总入口。

## 🎯 这是什么

这个仓库主要用来整理我在学习和输出 AI 过程中的内容，包括：

- **系统知识地图**：把 AI 相关主题按模块拆开，方便查找和持续更新
- **热门方向整理**：例如 RAG、Agent、MCP、工程落地
- **文章与案例**：把一部分热点拆解、教程和思考沉淀下来
- **长期资料入口**：方便统一访问，而不是资料散落在不同地方
