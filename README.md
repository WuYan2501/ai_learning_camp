## 💡 这是什么？

这不是简单的资料堆砌，而是一条**经过系统梳理的 AI 学习路径**：

- 📚 **完整知识体系** - 覆盖基础认知、模型原理、应用实践、工程落地
- 🎯 **热门方向聚焦** - RAG、Agent、MCP、多模态等前沿技术深度解析
- 📝 **实战案例沉淀** - 技术拆解、教程实操、面试准备一应俱全
- 🔄 **持续更新迭代** - 跟踪 AI 领域最新动态与技术演进

- 🚀 **实践者** - 想把 AI 能力真正落地到工作中

## 🚀 开始使用

### 方式一：在线阅读（推荐）

访问 **[https://wuyan2501.github.io/ai_learning_camp/](https://wuyan2501.github.io/ai_learning_camp/)** 获得最佳阅读体验

- ✅ 搜索功能
- ✅ 目录导航
- ✅ 响应式设计
- ✅ 暗色模式

### 方式二：本地浏览

```bash
# 克隆仓库
git clone https://github.com/WuYan2501/ai_learning_camp.git
cd ai_learning_camp

# 安装 docsify（可选）
npm i docsify-cli -g

# 启动本地服务器
docsify serve

# 或直接查看 Markdown 文件
```
